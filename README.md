## Sagemath 9 instructions
To test the implementation in Sagemath 9, pytest needs to be installed.
```bash
$ sage -sh
(sage-sh) $ pip install pytest
(sage-sh) $ exit
```

Tests can then be run using
```bash
$ sage -t test.py
```
